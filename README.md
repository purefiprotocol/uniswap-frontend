## Local Development

Install dependencies

```bash
yarn
```

Copy .env.sample file from .env

```bash
cp .env.sample .env
```

Provide VITE_WALLET_CONNECT_PROJECT_ID value in .env

```bash
VITE_WALLET_CONNECT_PROJECT_ID=********************************
```

Start dev server

```bash
yarn start
```
